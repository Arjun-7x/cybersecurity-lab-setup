# 🔐 Personal Cybersecurity Lab Setup

## 📌 Overview
This project documents my hands-on cybersecurity lab built using virtual machines to simulate real-world attack and defense scenarios.

## 🧰 Tools & Platforms
- Kali Linux
- Metasploitable 2
- VirtualBox
- DVWA (Damn Vulnerable Web App)

## 🎯 Objectives
- Practice penetration testing techniques
- Understand vulnerabilities and exploitation
- Build real-world cybersecurity skills

## 🖥️ Lab Architecture
- Host Machine → VirtualBox
- Kali Linux (Attacker)
- Metasploitable 2 (Target)

## 🚀 Labs Covered
- DVWA Setup & Configuration
- Command Injection
- Network Scanning (Nmap)
- Basic Exploitation

## ⚔️ Labs Performed
- DVWA Setup & Configuration
- Command Injection Testing
- Network Scanning using Nmap
- Basic Exploitation
